## 1.项目介绍

本项目是为了快速构建应用程序的说明文档而创建的，使用了VuePress theme hope主题进行了修改

+ 开发者只需要专注于编写文档，不需要关心文档的样式和布局。
+ 顶部的导航栏和侧边栏需要自己配置，参考已有的代码即可。

## 2.如何运行

安装依赖：

```shell
pnpm install
```

运行：

```shell
pnpm run docs:dev
```
